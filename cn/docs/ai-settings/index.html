AI 设置 ​

本文档将介绍如何使用 Chaterm 的 AI 功能，通过正确的设置，你可以充分利用 AI 助手提升工作效率。

工作模式 ​

Chaterm 提供了三种不同的 AI 工作模式，以满足不同场景的需求：

前置条件 ​

在开始使用 AI 功能之前，请确保您已经：

  • 在设置中完成 AI 配置
  • 测试 AI 连接状态正常

详细配置说明请参考 设置- AI 设置

Chat 模式 ​

  • 功能：与 AI 进行自然语言对话
  • 适用场景：
    • 知识学习和咨询
    • 头脑风暴
    • 技术讨论

IMPORTANT

注意：该模式下不支持终端操作

Command 模式 ​

  • 功能：在当前活跃终端中执行相关任务
  • 特点：所有的输入输出都在终端界面中显示
  • 适用场景：
    • 命令行操作
    • 本地任务执行

Agent 模式 ​

  • 功能：在任意主机上执行命令和任务
  • 特点：
    • 操作结果在 AI 内容栏中显示
    • 支持全自动执行
  • 适用场景：
    • 多主机任务处理
    • 系统问题排查
    • 自动化运维操作

模型设置 ​

1.选择 API 提供商 ​

Chaterm 目前支持以下 AI 服务提供商：

  • Amazon Bedrock
  • DeepSeek
  • OpenAI Compatible（兼容 OpenAI 接口的服务）

2. 填写认证信息 ​

根据你选择的 API 提供商，需要填写相应的认证信息：

  • Amazon Bedrock：需要配置 AWS 凭证
  • DeepSeek：需要填写 API Key
  • OpenAI Compatible：需要填写 API Key 和服务端点（Endpoint）

3. 选择模型 ​

在认证信息填写完成后，你可以：

  • 直接输入模型名称
  • 或从下拉列表中选择可用的模型

4. 验证配置 ​

完成以上设置后：

  1. 点击界面上的"Check"按钮
  2. 系统将自动测试连接
  3. 如果配置正确，将弹出成功提示
  4. 此时即可开始使用 AI 功能

IMPORTANT

注意：请确保填写的认证信息准确无误，建议定期检查 API 密钥的有效性，如遇到连接问题，请检查网络状态和服务商服务状态。
