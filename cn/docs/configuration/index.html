基本设置 ​

Chaterm 提供了一个灵活的配置系统，帮助您高效管理SSH连接和加密密钥。本指南将带您了解 Chaterm 的基本配置选项，以充分利用其功能。

主机设置 ​

Chaterm 的核心功能之一是通过 SSH 管理和连接到远程服务器。您可以通过 Chaterm 界面配置您的 SSH 连接。

添加主机 ​

在 Chaterm 中，每个SSH连接都表示为一个资产。 在侧边栏的本地连接中点击主机按钮添加资产，资产包括以下详细信息：

属性描述
别名连接的友好名称
连接地址服务器的主机名或IP
端口号SSH端口（默认：22）
认证类型基于密码或基于密钥
用户名SSH 登录用户名
分组信息分组标签信息

修改与删除 ​

  • 修改：在资产管理页面中，选择需要修改的主机卡片，点击编辑按钮进行修改。
  • 删除：在资产管理页面中，右键点击需要删除的主机卡片，在弹出菜单中选择删除选项。系统会要求二次确认，确认后即可删除该主机。

密钥设置 ​

为了与远程服务器进行安全认证，Chaterm 提供了一个强大的密钥管理系统。

密钥管理 ​

您的 SSH 密钥安全地存储在本地数据库中，并且可以被多个连接资产引用。这意味着您可以创建和存储密钥一次，然后在多个服务器连接中使用它们。

创建密钥 ​

1.点击左侧菜单中的 KeyChain 图标 2.在密钥管理页面点击"添加密钥"按钮 3.在弹出的表单中填写以下信息：

  • 密钥名称
  • 密钥内容
  • 其他相关配置

修改密钥 ​

要修改现有密钥的配置： 1.在密钥管理页面找到需要修改的密钥 2.右键点击该密钥卡片 3.在弹出的菜单中选择"编辑"选项 4.在编辑表单中更新相关信息

删除密钥 ​

删除不需要的密钥： 1.在密钥管理页面找到要删除的密钥 2.右键点击该密钥卡片 3.在弹出的菜单中选择"删除"选项 4.确认删除操作

个性化配置 ​

界面主题 ​

  • 支持暗色主题和亮色主题
  • 可根据个人喜好和使用环境选择合适的主题

终端显示 ​

  • 字体大小：自定义终端文字的显示大小
  • 回滚量：设置终端可以回滚查看的历史内容量
  • 光标样式：可选择不同的光标显示样式，适应个人使用习惯

语言设置 ​

  • 支持中文和英文两种语言
  • 切换后即时生效，无需重启应用

水印显示 ​

  • 可选择开启或关闭界面水印
  • 不影响终端的正常使用功能

使用建议 ​

  • 建议根据显示器尺寸调整合适的字体大小
  • 如需查看大量历史记录，可适当增加回滚量
  • 可以根据使用环境选择合适的主题，减少视觉疲劳
