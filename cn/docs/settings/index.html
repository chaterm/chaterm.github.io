系统设置 ​

本指南将帮助你管理和配置 Chaterm 的各项系统功能和扩展设置功能。通过设置你可以自定义终端的使用体验，提升工作效率。

系统设置 ​

界面主题 ​

  • 支持暗色主题和亮色主题
  • 可根据个人喜好和使用环境选择合适的主题

终端显示 ​

  • 字体大小：自定义终端文字的显示大小
  • 回滚量：设置终端可以回滚查看的历史内容量
  • 光标样式：可选择不同的光标显示样式，适应个人使用习惯

语言设置 ​

  • 支持中文和英文两种语言
  • 切换后即时生效，无需重启应用

水印显示 ​

  • 可选择开启或关闭界面水印
  • 不影响终端的正常使用功能

使用建议 ​

  • 建议根据显示器尺寸调整合适的字体大小
  • 如需查看大量历史记录，可适当增加回滚量
  • 可以根据使用环境选择合适的主题，减少视觉疲劳

扩展设置 ​

扩展可用扩展功能 ​

自动补全 ​

自动补全功能可以在你输入命令时提供智能提示，帮助你更快速地完成命令输入。

  • 开启后：命令输入过程中会显示建议的补全选项
  • 关闭后：需要完整手动输入命令

Vim 编辑器 ​

为喜欢使用 Vim 的用户提供可视化的 Vim 编辑模式。

  • 开启后：可以使用 Vim 快捷键和命令进行文本编辑
  • 关闭后：使用标准的文本编辑模式

全局 Alias ​

允许你设置和使用全局命令别名，简化常用命令的输入。

  • 开启后：可以使用已配置的命令别名
  • 关闭后：需要输入完整的命令

全局语法高亮 ​

为命令行界面提供语法高亮功能，提高代码和命令的可读性。

  • 开启后：命令、参数和输出将带有颜色标识
  • 关闭后：所有文本均以默认颜色显示

注意事项 ​

  • 所有扩展设置的修改将即时生效
  • 你可以随时根据需要开启或关闭这些功能
  • 建议根据个人使用习惯和工作需求来配置这些扩展
