Account Management ​

The account management feature provides complete account lifecycle management, including registration, login, security, and comprehensive management.

Account Registration ​

Email Registration ​

Register an account using an email address with verification code verification.

Third-party Authentication ​

International Platforms ​

  • Google - Quick login using Google account
  • GitHub - Quick login using GitHub account

Domestic Platforms ​

  • QQ - Quick login using QQ account

Account Information Modification ​

You can modify the following personal information:

  • Display name
  • Username
  • Phone number
  • Login password

Abnormal Login Detection ​

The system automatically detects the following situations:

  • New login locations
  • New device logins

When abnormal login is detected, the system will:

  1. Send security notifications
  2. Trigger secondary verification
  3. Require identity confirmation

Security Reminder

If you discover abnormal login activity, please change your password immediately.
