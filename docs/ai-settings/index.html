AI Settings ​

This document will introduce how to use Chaterm's AI features. With the correct settings, you can make full use of the AI assistant to improve your work efficiency.

Working Mode ​

Chaterm provides three different AI working modes to meet the needs of different scenarios:

Prerequisites ​

Before starting to use the AI function, make sure you have:

  • Complete AI configuration in Settings
  • Test AI connection status is normal

For detailed configuration instructions, please refer to Settings - AI Settings

Chat Mode ​

  • Function: Have natural language conversations with AI
  • Applicable scenarios:
  • Knowledge learning and consultation
  • Brainstorming
  • Technical discussion

IMPORTANT

Note: Terminal operations are not supported in this mode

Command Mode ​

  • Function: Execute related tasks in the current active terminal
  • Features: All input and output are displayed in the terminal interface
  • Applicable scenarios:
  • Command line operation
  • Local task execution

Agent Mode ​

  • Function: Execute commands and tasks on any host
  • Features:
  • Operation results are displayed in the AI content bar
  • Supports fully automatic execution
  • Applicable scenarios:
  • Multi-host task processing
  • System troubleshooting
  • Automated operation and maintenance

Model settings ​

1. Select an API provider ​

Chaterm currently supports the following AI service providers:

  • Amazon Bedrock
  • DeepSeek
  • OpenAI Compatible (services compatible with OpenAI interfaces)

2. Fill in authentication information ​

Depending on the API provider you choose, you need to fill in the corresponding authentication information:

  • Amazon Bedrock: AWS credentials need to be configured
  • DeepSeek: API Key needs to be filled in
  • OpenAI Compatible: API Key and service endpoint need to be filled in

3. Select a model ​

After completing the authentication information, you can:

  • Enter the model name directly
  • Or select an available model from the drop-down list

4. Verify the configuration ​

After completing the above settings:

  1. Click the "Check" button on the interface
  2. The system will automatically test the connection
  3. If the configuration is correct, a success prompt will pop up
  4. You can start using AI features at this time

IMPORTANT

Note: Please ensure that the authentication information you enter is correct. It is recommended that you check the validity of the API key regularly. If you encounter connection problems, please check the network status and service provider service status.
